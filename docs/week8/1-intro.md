# 1. Introduction

This week's activities cover:

- Full stack: accessing the backend routes (FastAPI app) from the frontend app (Streamlit app is used as the example)
- Configuring the app using PydanticSettings and .env file
- Error logging and handling in FastAPI
- Optional: authentication for the full stack

The activities assume that you completed the week 6 and 7 activities and are using code resembling 
the [tutor solution](https://github.com/nicholsons/comp0034_tutorialapp_fastapi/tree/week7).
